# Coq_Kruispunt
Coq script. 

# Setup

1. Clone Repository
> git clone https://github.com/jhaasdijk/Coq_Kruispunt.git

2. Fork Repository

3. Add remote branches
> git remote add ... ...

    * Jasper's version
    > git remote add jasper https://github.com/jhaasdijk/Coq_Kruispunt.git

    * Own version
    > git remote add name ...

# Workflow

1. Fetch newest version
> git fetch jasper

2. Merge fetched version
> git merge jasper

3. Change something

4. Add changes
> git add ...	
> git add Coq_script

5. Commit changes
> git commit

6. Push changes to your repo
> git push ... ...	
> git push github master

7. Create pull request
